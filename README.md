# Pip Pip – Insect‑Corridor Mapper

A lightweight, free web app that lets anyone:

1. **Draw a straight “insect corridor”** on a map of Noida (or any location).  
2. **Generate a two‑page PDF ordinance** that can be submitted to the municipal authority.  
3. **See the corridor’s length** and a list of its coordinates.

## Tech stack (all free)

- **HTML / CSS / JavaScript** – static site, no backend.  
- **Leaflet** – interactive map.  
- **Leaflet‑Draw** – drawing toolbar.  
- **jsPDF + html2canvas** – convert a hidden HTML template into a PDF.  
- **GitHub Pages** – free hosting (`https://your‑username.github.io/pippip/`).
